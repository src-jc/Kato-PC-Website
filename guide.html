Step-by-Step Build Guide

1

Prepare Your Workspace

Clear a large, clean surface. Unbox all components and verify everything is present. Read manuals for motherboard and case.

2

Install CPU on Motherboard

Open the CPU socket lever, align the CPU (match the triangle/arrow), gently place it, and close the lever. Never force it!

3

Install CPU Cooler

Apply thermal paste if needed (usually pre-applied). Follow cooler instructions carefully. Ensure it's securely mounted.

4

Install RAM

Check motherboard manual for optimal slots (usually A2 and B2 for 2 sticks). Open clips, align notch, press down until clips snap.

5

Install M.2 SSD (if applicable)

Locate M.2 slot, remove screw, insert SSD at 30¬∞ angle, push down, secure with screw. Some motherboards have heatsinks.

6

Install Motherboard I/O Shield

Press I/O shield into case from inside. Ensure it clicks in place on all sides.

7

Install Motherboard in Case

Install standoffs in correct positions. Carefully lower motherboard, align with I/O shield and standoffs. Screw in securely.

8

Install Power Supply

Position PSU with fan facing down (if case has vent) or to the side. Secure with 4 screws from the back.

9

Connect Front Panel Connectors

Refer to motherboard manual. Connect power button, reset, HDD LED, and power LED. USB and audio headers too.

10

Connect Power Cables

Connect 24-pin ATX, 8-pin CPU power, SATA power (if needed), and PCIe power for GPU. Ensure all connections are firm.

11

Install Graphics Card

Remove PCIe slot covers. Insert GPU into top PCIe x16 slot. Press firmly until it clicks. Secure with screws. Connect power cables.

12

Cable Management & Final Check

Route cables behind motherboard tray. Use cable ties. Double-check all connections. Close side panel.

BIOS Setup

Accessing BIOS

Press Delete, F2, or F12 during boot (check motherboard manual).

Important BIOS Settings

  • Boot Order: Set your OS installation drive first
  • XMP/DOCP: Enable for RAM to run at advertised speed
  • SATA Mode: Set to AHCI for SSDs
  • Secure Boot: May need to disable for some OS installations
  • Fan Curves: Adjust for optimal cooling/noise balance
  • Date & Time: Set correctly for Windows activation

Operating System Installation

Windows 11 Installation

1

Create Installation Media

Download Windows Media Creation Tool from Microsoft. Create bootable USB drive (8GB minimum).

2

Boot from USB

Insert USB, restart PC, enter BIOS, set USB as first boot device. Save and exit.

3

Install Windows

Follow on-screen prompts. Select Custom installation, choose drive, format if needed. Windows will install and restart.

4

Initial Setup

Complete Windows setup: Region, keyboard, Microsoft account, privacy settings.

Cable Management Tips

Route Behind Motherboard Tray

Use the space behind the motherboard to hide cables. Most modern cases have cutouts for this purpose.

Use Cable Ties

Bundle cables together with velcro straps or zip ties. Don't overtighten to allow future modifications.

Plan Your Routes

Think about cable paths before connecting. Shortest route isn't always the cleanest.

Use Modular PSU

Only connect cables you need. Reduces clutter significantly.

Common Issues & Solutions

PC Won't Turn On

  • Check PSU switch is on (I position)
  • Verify power cable is plugged in
  • Ensure 24-pin and 8-pin CPU power connected
  • Check front panel connectors

No Display Output

  • Connect monitor to GPU, not motherboard
  • Check GPU power cables connected
  • Reseat RAM (try one stick at a time)
  • Clear CMOS (remove battery for 30 seconds)

High Temperatures

  • Check CPU cooler is properly mounted
  • Verify thermal paste application
  • Ensure fans are spinning
  • Check case airflow (front intake, rear exhaust)

RAM Not Detected

  • Ensure RAM is fully seated (should click)
  • Try different slots (check manual)
  • Test one stick at a time
  • Enable XMP in BIOS

Regular Checks

  • Check for Windows updates
  • Update GPU drivers
  • Clean dust filters
Every 3 Months

Deep Cleaning

  • Power off and unplug PC
  • Use compressed air on fans
  • Clean heatsinks and radiators
  • Check cable connections
Annually

Major Maintenance

  • Consider reapplying thermal paste
  • Check all screws and mounting
  • Update BIOS if needed
  • Backup important data
